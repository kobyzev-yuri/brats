# Перезапуск n8n с отключенным Basic Auth

## Если n8n уже запущен

1. **Остановите текущий контейнер:**
```bash
docker stop n8n-brats
docker rm n8n-brats
```

2. **Запустите заново:**
```bash
cd /projects/brats/n8n
./start_n8n.sh
```

Или через docker-compose:
```bash
cd /projects/brats/n8n
docker-compose down
docker-compose up -d
```

3. **Откройте n8n в браузере:**
   - URL: http://localhost:5678
   - Теперь не будет запроса на basic auth
   - При первом входе создайте учетную запись администратора (email + пароль)

## Создание первого пользователя

При первом входе n8n попросит создать учетную запись:
- **Email**: ваш email (например: `admin@localhost` или `your@email.com`)
- **Password**: ваш пароль
- **First Name**: ваше имя
- **Last Name**: ваша фамилия

После создания учетной записи вы сможете входить в n8n используя email и пароль.

## Примечание

- Basic Auth отключен для удобства локальной разработки
- Для продакшена рекомендуется включить Basic Auth обратно
- Данные n8n сохраняются в `~/.n8n/`, поэтому учетная запись сохранится после перезапуска
