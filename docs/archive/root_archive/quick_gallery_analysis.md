# Быстрая инструкция: Анализ галереи изображений

## Краткий алгоритм для Gemini

```
1. browser_navigate(URL с вкладкой)
2. browser_wait_for(3 сек)
3. browser_snapshot()
4. Найти "X из Y" → определить количество изображений
5. Для каждого изображения:
   - Извлечь URL из snapshot
   - Проанализировать через Gemini Vision
   - Сохранить {index, url, description}
   - Кликнуть "Следующий слайд" (если не последнее)
6. Вернуть все результаты
```

## Промпт для Gemini Vision

```
Проанализируй это изображение:
1. Что изображено? (интерьер/экстерьер/план)
2. Какие помещения/зоны?
3. Стиль оформления?
4. Материалы и отделка?
5. Детали интерьера/мебели?
6. Настроение и впечатление?
7. Особенности?

Опиши детально для потенциального покупателя.
```

## Формат результата

```json
{
  "tab": "STANDARD",
  "total_images": 5,
  "images": [
    {"index": 1, "url": "...", "description": "..."},
    {"index": 2, "url": "...", "description": "..."}
  ]
}
```

## Важно

- ✅ Используй browser инструменты (не web_search)
- ✅ Жди загрузки после каждого клика (1-2 сек)
- ✅ Анализируй ТОЛЬКО изображения галереи
- ✅ Игнорируй логотипы и иконки
