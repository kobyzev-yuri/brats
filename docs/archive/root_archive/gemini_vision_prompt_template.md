# Шаблоны промптов для анализа изображений через Gemini Vision

## Базовый промпт для описания изображения

```
Проанализируй это изображение и опиши его детально:

1. Что изображено на фото? (интерьер, экстерьер, план, деталь и т.д.)
2. Какие помещения/зоны видны?
3. Какой стиль оформления?
4. Какие материалы и отделка использованы?
5. Какие детали интерьера/мебели присутствуют?
6. Какое настроение создает изображение?
7. Какие особенности или уникальные элементы?

Опиши максимально подробно, как будто описываешь для потенциального покупателя.
```

## Детальный промпт для интерьера

```
Детально опиши это изображение интерьера:

КОНТЕКСТ: Это изображение из каталога домов с отделкой STANDARD.

ОПИСАНИЕ ДОЛЖНО ВКЛЮЧАТЬ:

1. Тип помещения:
   - Какая комната изображена? (гостиная, кухня, спальня, ванная и т.д.)
   - Какие зоны видны?

2. Стиль и дизайн:
   - Какой стиль интерьера? (минимализм, классика, современный и т.д.)
   - Какая цветовая гамма?
   - Какое общее впечатление?

3. Отделка и материалы:
   - Стены: какой материал, цвет, текстура?
   - Пол: какой материал, цвет, покрытие?
   - Потолок: какой тип, цвет?
   - Окна: размер, тип, наличие штор/жалюзи?

4. Мебель и детали:
   - Какая мебель присутствует?
   - Какие декоративные элементы?
   - Освещение: какие светильники видны?

5. Особенности:
   - Что делает это помещение привлекательным?
   - Какие уникальные детали?
   - Какое качество отделки видно?

6. Настроение:
   - Какое впечатление создает интерьер?
   - Уютный, просторный, современный и т.д.?

Опиши так, чтобы потенциальный покупатель мог представить себя в этом пространстве.
```

## Промпт для экстерьера

```
Детально опиши это изображение экстерьера дома:

КОНТЕКСТ: Это изображение из каталога домов с отделкой STANDARD.

ОПИСАНИЕ ДОЛЖНО ВКЛЮЧАТЬ:

1. Тип изображения:
   - Фасад дома?
   - Вид со двора?
   - Общий план участка?

2. Архитектура:
   - Стиль дома (современный, классический и т.д.)
   - Количество этажей
   - Форма крыши
   - Материалы фасада

3. Детали:
   - Окна: размер, форма, расположение
   - Двери: тип, расположение
   - Балконы, террасы, крыльцо
   - Декоративные элементы

4. Окружение:
   - Что видно вокруг дома?
   - Благоустройство территории
   - Забор, ворота
   - Озеленение

5. Качество:
   - Качество отделки фасада
   - Общее впечатление от дома
   - Привлекательность для покупателя

Опиши так, чтобы потенциальный покупатель мог представить этот дом.
```

## Промпт для планов и схем

```
Опиши этот план/схему помещения:

КОНТЕКСТ: Это план из каталога домов с отделкой STANDARD.

ОПИСАНИЕ ДОЛЖНО ВКЛЮЧАТЬ:

1. Тип плана:
   - План какого этажа? (первый, второй)
   - План какого помещения? (общий план, детальный план комнаты)

2. Расположение помещений:
   - Какие комнаты видны?
   - Как они расположены относительно друг друга?
   - Какие проходы и коридоры?

3. Размеры:
   - Какие размеры указаны?
   - Какая общая площадь?
   - Размеры отдельных помещений?

4. Особенности планировки:
   - Удобство расположения комнат
   - Функциональность планировки
   - Преимущества и особенности

5. Детали:
   - Расположение окон и дверей
   - Лестницы (если есть)
   - Санузлы, кухня, спальни

Опиши планировку так, чтобы потенциальный покупатель мог понять удобство и функциональность.
```

## Промпт для технических деталей

```
Опиши эту техническую деталь или элемент отделки:

КОНТЕКСТ: Это изображение детали из каталога домов с отделкой STANDARD.

ОПИСАНИЕ ДОЛЖНО ВКЛЮЧАТЬ:

1. Что изображено:
   - Какой элемент отделки?
   - Какая техническая деталь?

2. Материалы:
   - Из какого материала изготовлено?
   - Качество материала?

3. Детали:
   - Размеры (если видны)
   - Цвет, текстура
   - Способ монтажа

4. Качество:
   - Качество исполнения
   - Внимание к деталям
   - Профессионализм отделки

5. Назначение:
   - Для чего используется?
   - Какую функцию выполняет?

Опиши так, чтобы показать качество и внимание к деталям при отделке.
```

```
Определи категорию этого изображения и кратко опиши:

Категории:
- interior (интерьер)
- exterior (экстерьер)
- plan (план/схема)
- detail (деталь отделки)
- construction (строительство)

Ответ в формате:
Категория: [категория]
Описание: [краткое описание]
```

## Использование в коде

### Python с Gemini API

```python
import google.generativeai as genai

def analyze_image(image_url: str, prompt: str) -> str:
    """Анализирует изображение через Gemini Vision"""
    genai.configure(api_key="YOUR_API_KEY")
    model = genai.GenerativeModel('gemini-pro-vision')
    
    response = model.generate_content([
        prompt,
        image_url  # или загруженное изображение
    ])
    
    return response.text

# Использование
prompt = """
Проанализируй это изображение и опиши его детально:
1. Что изображено на фото?
2. Какой стиль оформления?
3. Какие материалы использованы?
...
"""

description = analyze_image("https://example.com/image.jpg", prompt)
```

### JavaScript/TypeScript

```javascript
async function analyzeImage(imageUrl, prompt) {
    const response = await fetch('https://generativelanguage.googleapis.com/v1beta/models/gemini-pro-vision:generateContent', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
            'x-goog-api-key': 'YOUR_API_KEY'
        },
        body: JSON.stringify({
            contents: [{
                parts: [
                    { text: prompt },
                    { 
                        inline_data: {
                            mime_type: "image/jpeg",
                            data: await fetchImageAsBase64(imageUrl)
                        }
                    }
                ]
            }]
        })
    });
    
    const data = await response.json();
    return data.candidates[0].content.parts[0].text;
}
```

## Рекомендации

1. **Используйте детальные промпты** для лучших результатов
2. **Указывайте контекст** (STANDARD, BLACK BOX и т.д.) в промпте
3. **Задавайте конкретные вопросы** для структурированного ответа
4. **Адаптируйте промпт** под тип изображения (интерьер/экстерьер/план)
5. **Используйте категоризацию** для организации результатов
