# Системный промпт для работы с вкладками

## Краткая версия для системного промпта

```
ПРАВИЛО ДЛЯ URL С ВКЛАДКАМИ:

Когда пользователь предоставляет URL вида:
https://innovatory-club.ru/katalog#!/tab/1063728081-X

1. Хэш #!/tab/1063728081-X указывает на конкретную вкладку:
   - X=1 → BLACK BOX
   - X=2 → WHITE BOX
   - X=3 → STANDARD
   - X=4 → DESIGN

2. Клиент видит ТОЛЬКО одну вкладку, не все сразу.

3. ВСЕГДА используй browser_navigate + browser_wait_for(3 сек) + browser_snapshot
   НЕ используй web_search для таких URL.

4. Парси ТОЛЬКО контент активной вкладки X.
   ИГНОРИРУЙ контент других вкладок.

5. Проверь правильность по индикаторам активной вкладки перед парсингом.
```

## Расширенная версия для системного промпта

```
КОНТЕКСТ: Работа с динамическими вкладками на страницах

ПРАВИЛА:

1. URL с хэш-фрагментом #!/tab/1063728081-X указывает на конкретную вкладку:
   - Клиент видит ТОЛЬКО эту вкладку в браузере
   - Остальные вкладки скрыты или неактивны

2. ОБЯЗАТЕЛЬНО используй браузерные инструменты:
   - browser_navigate(url)
   - browser_wait_for(time: 3) - критически важно для выполнения JS
   - browser_snapshot() - для получения актуального состояния
   - При необходимости: browser_click() для переключения вкладки

3. НЕ используй web_search для таких URL - он парсит статический HTML 
   до выполнения JavaScript и всегда возвращает первую вкладку.

4. После получения снимка страницы:
   - Определи активную вкладку по индикаторам
   - Извлеки ТОЛЬКО её контент
   - Игнорируй контент других вкладок, даже если он в DOM

5. Индикаторы вкладок:
   - STANDARD (X=3): "Просторный, уютный дом", "более 300 домов", "8 950 000 Р"
   - BLACK BOX (X=1): "черновой ремонт включает", "8 350 000 Р"
   - WHITE BOX (X=2): "предчистовой ремонт включает", "8 950 000 Р"
   - DESIGN (X=4): "дизайнерский ремонт включает", "Дизайн-проект", "10 450 000 Р"

6. КРИТИЧЕСКАЯ ОШИБКА: Если в результате смешан контент разных вкладок - 
   это означает неправильный парсинг. Клиент видит только одну вкладку.

7. Перед возвратом результата проверь:
   - Соответствует ли результат запрошенной вкладке?
   - Нет ли контента других вкладок?
   - Правильны ли индикаторы активной вкладки?
```

## Минимальная версия (одна строка)

```
Для URL с #!/tab/1063728081-X: используй browser_navigate+wait_for(3)+snapshot, парси ТОЛЬКО вкладку X, игнорируй остальные.
```
