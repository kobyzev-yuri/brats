# Краткая инструкция: Выделение смысловых блоков (универсальный)

## Правило для Gemini

```
При парсинге любой веб-страницы извлекай ТОЛЬКО смысловые блоки:

ИЗВЛЕКАТЬ:
✅ Заголовки (H1-H6) из основного контента
✅ Описательные тексты (параграфы, абзацы)
✅ Списки (маркированные, нумерованные) с релевантной информацией
✅ Таблицы с данными
✅ Цены и стоимость (если есть)
✅ Галерея изображений (> 100x100px)
✅ Характеристики, преимущества, особенности
✅ Контакты (адрес, телефон, email, график)

ИГНОРИРОВАТЬ:
❌ Навигация (role: navigation)
❌ Хедер (role: banner)
❌ Футер (role: contentinfo)
❌ Формы (если не нужны)
❌ Кнопки навигации
❌ Иконки и декоративные элементы (< 100x100px)
❌ Служебные тексты: "Skip to main content", "Закрыть", "Навигационное меню"

Правила:
- Извлекай только из role: main, article, section
- Игнорируй элементы вне основного контента
- Фильтруй изображения по размеру (> 100x100px)
- Структурируй результат в JSON по типам блоков
- Адаптируйся под структуру любой страницы
```

## Формат результата

```json
{
  "page_url": "https://example.com/page",
  "page_title": "Название страницы",
  "semantic_blocks": [
    {
      "type": "heading",
      "level": 1,
      "text": "Основной заголовок"
    },
    {
      "type": "text",
      "content": "Описательный текст..."
    },
    {
      "type": "list",
      "title": "Заголовок списка",
      "items": ["Элемент 1", "Элемент 2"]
    },
    {
      "type": "price",
      "value": "1000",
      "currency": "RUB",
      "text": "1 000 руб"
    },
    {
      "type": "gallery",
      "images": [...]
    },
    {
      "type": "contacts",
      "address": "...",
      "phone": "...",
      "email": "..."
    }
  ],
  "ignored_blocks": ["Навигация", "Хедер", "Футер"]
}
```

## Ключевые слова и паттерны для идентификации

- **Заголовки**: элементы H1-H6 в основном контенте
- **Описание**: параграфы (p) и текстовые блоки после заголовков
- **Списки**: ul, ol элементы с релевантным содержимым
- **Таблицы**: table элементы с данными
- **Стоимость**: паттерны цен (цифры с валютой: "руб", "$", "€", "₽")
- **Контакты**: заголовки "Контакты", "Contact", паттерны телефонов (+X), email (@), адреса
- **Характеристики**: заголовки "Характеристики", "Преимущества", "Features", "Specifications"
