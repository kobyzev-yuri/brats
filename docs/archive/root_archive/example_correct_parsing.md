# Пример правильной обработки запроса с вкладками

## Запрос пользователя

```
При вводе url https://innovatory-club.ru/katalog#!/tab/1063728081-3
```

## Правильная последовательность действий для Gemini

### Шаг 1: Анализ URL
```
URL: https://innovatory-club.ru/katalog#!/tab/1063728081-3
Хэш-фрагмент: #!/tab/1063728081-3
Номер вкладки: 3
Тип вкладки: STANDARD (стандартный ремонт)
```

### Шаг 2: Использование браузерных инструментов

```javascript
// 1. Навигация на страницу
browser_navigate("https://innovatory-club.ru/katalog#!/tab/1063728081-3")

// 2. Ожидание выполнения JavaScript (критически важно!)
browser_wait_for(time: 3)

// 3. Получение снимка страницы
browser_snapshot()
```

### Шаг 3: Проверка активной вкладки

После получения снимка проверь наличие индикаторов STANDARD:
- ✅ "STANDARD (стандартный ремонт)" в названии вкладки
- ✅ "Просторный, уютный дом с качественным ремонтом" в тексте
- ✅ "более 300 домов" в описании
- ✅ "Коттедж со стандартным ремонтом" в кнопке резервирования

Если видишь индикаторы других вкладок (например, "черновой ремонт" или "8 350 000 Р") - 
это означает, что вкладка не переключилась. Нужно явно кликнуть на STANDARD.

### Шаг 4: Парсинг ТОЛЬКО контента STANDARD

**Извлеки ТОЛЬКО:**
- Описание STANDARD: "Просторный, уютный дом с качественным ремонтом..."
- Что включает STANDARD: список для стандартного ремонта
- Стоимость STANDARD: "8 950 000 Р"
- Кнопка резервирования STANDARD

**НЕ извлекай:**
- ❌ Информацию о BLACK BOX (8 350 000 Р, "черновой ремонт")
- ❌ Информацию о WHITE BOX (8 950 000 Р, "предчистовой ремонт")
- ❌ Информацию о DESIGN (10 450 000 Р, "дизайнерский ремонт")

### Шаг 5: Форматирование результата

```
STANDARD (стандартный ремонт)

Общая площадь: 140 м² (1 этаж – 80 м², 2 этаж – 60 м²)

Описание:
Просторный, уютный дом с качественным ремонтом, в который можно заселиться 
сразу после покупки. Мы построили более 300 домов и точно знаем, как сделать 
ремонт так, чтобы он радовал вас долгие годы. STANDARD — проверенное временем 
решение, сочетающее практичность, долговечность и стиль: минималистичная 
классика, которая не выходит из моды.

Что включает:
[список только для стандартного ремонта]

Стоимость: 8 950 000 Р
```

## Примеры неправильной обработки

### ❌ ОШИБКА 1: Смешанный контент всех вкладок

```
Результат:
- BLACK BOX: 8 350 000 Р
- WHITE BOX: 8 950 000 Р  
- STANDARD: 8 950 000 Р
- DESIGN: 10 450 000 Р
```
**Проблема**: Пользователь запросил только STANDARD, но получил все вкладки

### ❌ ОШИБКА 2: Использование веб-поиска вместо браузера

```
Использован: web_search("https://innovatory-club.ru/katalog#!/tab/1063728081-3")
Результат: Всегда возвращает BLACK BOX (первая вкладка)
```
**Проблема**: Веб-поиск не выполняет JavaScript, поэтому не видит переключение вкладки

### ❌ ОШИБКА 3: Недостаточное ожидание

```
browser_navigate(...)
browser_wait_for(time: 0.5)  // Слишком мало!
browser_snapshot()
Результат: Вкладка еще не переключилась
```
**Проблема**: JavaScript не успел выполниться

## Правильный промпт для Gemini

Когда пользователь предоставляет такой URL, используй этот промпт:

```
Пользователь запросил информацию по URL с вкладкой STANDARD (номер 3).

ВАЖНО: 
- Используй browser_navigate + browser_wait_for(3) + browser_snapshot
- Парси ТОЛЬКО контент вкладки STANDARD
- Игнорируй контент BLACK BOX, WHITE BOX, DESIGN
- Клиент видит только одну вкладку, не все сразу

Проверь индикаторы STANDARD:
- "Просторный, уютный дом с качественным ремонтом"
- "более 300 домов"
- "Коттедж со стандартным ремонтом"

Если видишь другие индикаторы - вкладка не переключилась, нужно кликнуть на STANDARD.
```

## Чек-лист для Gemini

Перед возвратом результата проверь:

- [ ] Определен номер вкладки из URL (3 = STANDARD)
- [ ] Использованы браузерные инструменты (не веб-поиск)
- [ ] Выполнено ожидание минимум 3 секунды
- [ ] Проверена активная вкладка по индикаторам
- [ ] Извлечен ТОЛЬКО контент STANDARD
- [ ] НЕ включен контент других вкладок
- [ ] Результат содержит только информацию о STANDARD

## Резюме

**Ключевой момент**: 
URL с `#!/tab/1063728081-3` означает, что клиент видит ТОЛЬКО вкладку STANDARD.
Парси ТОЛЬКО её контент, игнорируя остальные вкладки, даже если они присутствуют в DOM.
