# Быстрая инструкция: Вторичная обработка блоков

## Что это?

Функция для доработки уже существующих блоков в KB с помощью Gemini по вашим дополнительным инструкциям.

## Как использовать

### 1. Выберите блоки
- В списке блоков: отметьте чекбоксы нужных блоков
- Или на странице деталей блока: нажмите кнопку "Вторичная обработка"

### 2. Введите инструкции
Опишите, что нужно изменить/дополнить:

**Примеры:**
- "Добавь больше деталей о материалах и технологиях"
- "Улучши описание, сделай его более привлекательным"
- "Добавь информацию о гарантии и сроках"
- "Структурируй текст лучше, добавь подзаголовки"
- "Оптимизируй для SEO, сохраняя естественность"

### 3. Отправьте на обработку
Gemini обработает блоки и автоматически обновит их в KB.

## Где найти

- **В списке блоков**: Кнопка "Вторичная обработка" в массовых операциях
- **На странице блока**: Кнопка с иконкой повтора
- **В меню**: Пункт "Вторичная обработка"

## Требования

- Установлен `kb-service`
- Настроен `GEMINI_API_KEY` в `kb-service/config.env`

## Что происходит

1. Блоки отправляются в Gemini с вашими инструкциями
2. Gemini обрабатывает каждый блок
3. Обновленные блоки сохраняются в KB
4. Вы видите результат обработки

## Примеры инструкций

### Детализация
```
Добавь больше деталей о материалах и технологиях, используемых в отделке. 
Опиши преимущества каждого материала.
```

### Улучшение текста
```
Улучши описание, сделай его более привлекательным для покупателей. 
Добавь эмоциональную окраску, но сохрани факты.
```

### Дополнение информации
```
Добавь информацию о гарантии, сроках выполнения работ и возможности 
дополнительных услуг.
```

### Структурирование
```
Структурируй текст лучше: добавь подзаголовки, списки, выдели ключевые моменты.
Сделай текст более читаемым.
```

### SEO-оптимизация
```
Добавь ключевые слова для SEO (ремонт, отделка, дом, коттедж), 
но сохрани естественность и читаемость текста.
```
