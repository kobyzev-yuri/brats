# Быстрое исправление: Вторичная обработка не отображается

## ✅ Все файлы на месте, код правильный!

Проблема скорее всего в том, что Flask приложение не было перезапущено после изменений.

## 🔧 Решение (выполните по порядку):

### Шаг 1: Остановите Flask приложение
Если приложение запущено, нажмите `Ctrl+C` в терминале где оно работает.

### Шаг 2: Перезапустите приложение
```bash
cd /projects/brats
python3 kb_admin_app.py
```

### Шаг 3: Очистите кэш браузера
- **Chrome/Edge**: `Ctrl+Shift+R` (Windows/Linux) или `Cmd+Shift+R` (Mac)
- **Firefox**: `Ctrl+F5` или `Ctrl+Shift+R`
- Или откройте страницу в режиме инкогнито

### Шаг 4: Проверьте наличие кнопок

**В списке блоков** (`http://localhost:5000/blocks`):
- Должна быть **синяя кнопка "Вторичная обработка"** с иконкой повтора в разделе массовых операций (над таблицей)
- У каждого блока в колонке "Действия" должна быть **синяя кнопка с иконкой повтора**

**В меню** (левая панель):
- Должен быть пункт **"Вторичная обработка"** с иконкой повтора

**На странице деталей блока** (`http://localhost:5000/blocks/<id>`):
- Должна быть **синяя кнопка "Вторичная обработка"** рядом с другими действиями

### Шаг 5: Если кнопки все еще не видны

1. **Откройте консоль браузера** (F12 -> Console)
2. **Проверьте на ошибки JavaScript**
3. **Попробуйте открыть напрямую**: `http://localhost:5000/reprocess`

## 🧪 Проверка работоспособности

Запустите тестовый скрипт:
```bash
python3 test_reprocess.py
```

Если все проверки пройдены ✅, значит проблема только в перезапуске/кэше.
