# Инструкции: Выделение смысловых блоков на странице (универсальный)

## Задача

При парсинге любой веб-страницы нужно выделить блоки с независимым смыслом, игнорируя служебную информацию (навигация, меню, формы, декоративные элементы).

## Типы блоков на странице

### Смысловые блоки (нужно извлекать):

1. **Заголовки**
   - Основной заголовок страницы (H1)
   - Заголовки разделов (H2, H3, H4, H5, H6)
   - Подзаголовки и названия секций

2. **Описательные тексты**
   - Основной текст страницы
   - Параграфы и абзацы
   - Характеристики, преимущества, особенности

3. **Списки**
   - Маркированные списки (ul)
   - Нумерованные списки (ol)
   - Списки характеристик, преимуществ, особенностей

4. **Таблицы**
   - Таблицы с данными
   - Сравнительные таблицы
   - Таблицы характеристик

5. **Цены и стоимость**
   - Цены товаров/услуг
   - Стоимость в любой валюте
   - Ценовые диапазоны

6. **Галерея изображений**
   - Фотографии товаров/услуг
   - Иллюстрации
   - Диаграммы и схемы (если релевантны)

7. **Характеристики и особенности**
   - Технические характеристики
   - Преимущества
   - Особенности продукта/услуги

8. **Контакты**
   - Адрес
   - Телефон
   - Email
   - График работы
   - Социальные сети (если релевантно)

### Служебные блоки (нужно игнорировать):

1. **Навигационное меню**
   - Верхнее меню с ссылками
   - Боковое меню
   - Хлебные крошки

2. **Хедер**
   - Логотип
   - Кнопки "ОТПРАВИТЬ ЗАПРОС", "В КАТАЛОГ"
   - Телефон в шапке

3. **Футер**
   - Нижнее меню
   - Копирайт
   - Ссылки на политику конфиденциальности

4. **Формы**
   - Формы резервирования (если не нужны)
   - Формы обратной связи
   - Формы расчета ипотеки

5. **Декоративные элементы**
   - Иконки
   - Декоративные изображения
   - Фоновые элементы

6. **Кнопки навигации**
   - "Предыдущий слайд" / "Следующий слайд"
   - Кнопки закрытия модальных окон

7. **Скрипты и метаданные**
   - Счетчики аналитики
   - Метатеги
   - Скрытые элементы

## Алгоритм выделения блоков

### Шаг 1: Идентификация ролей элементов

В browser_snapshot() ищи элементы по ролям:

```javascript
// Смысловые блоки
- role: main, article, section
- role: heading (h1, h2, h3)
- role: list (для списков преимуществ, что включает)
- role: img (только изображения контента, не иконки)

// Служебные блоки (игнорировать)
- role: banner (хедер)
- role: navigation (меню)
- role: contentinfo (футер)
- role: dialog (модальные окна, если не нужны)
- role: form (формы, если не нужны)
- role: button (кнопки навигации, закрытия)
```

### Шаг 2: Фильтрация по содержимому

Используй ключевые слова и паттерны для идентификации блоков:

**Заголовки:**
- Искать элементы H1-H6 в основном контенте
- Игнорировать заголовки в навигации, футере

**Описание:**
- Искать параграфы (p) и текстовые блоки после заголовков
- Искать div, section с текстовым содержимым
- Исключать текст из навигации, футера, форм

**Списки:**
- Искать ul, ol элементы с релевантным содержимым
- Игнорировать списки в навигации, футере
- Определять тип списка по контексту (характеристики, преимущества, особенности)

**Таблицы:**
- Искать table элементы с данными
- Исключать таблицы для верстки (если есть)

**Стоимость:**
- Искать паттерны цен: цифры с валютой ("руб", "₽", "$", "€", "£" и т.д.)
- Искать текст: "Стоимость", "Цена", "Price", "Cost"
- Искать паттерны: "X XXX XXX", "X'XXX'XXX", "X,XXX.XX"

**Галерея:**
- Искать контейнеры с изображениями (img, picture, figure)
- Исключать логотипы (маленькие размеры, в хедере)
- Исключать иконки (маленькие размеры, декоративные)
- Извлекать alt-текст и описания изображений

**Контакты:**
- Искать заголовки: "Контакты", "Contact", "Contact Us", "Связаться"
- Искать паттерны: телефоны (+X ...), email (@), адреса
- Искать график работы: "График", "Schedule", "Hours"

**Характеристики:**
- Искать заголовки: "Характеристики", "Преимущества", "Features", "Specifications", "Benefits"
- Искать списки после этих заголовков

## Правила фильтрации

### Игнорировать элементы, если:

1. **Роль = navigation**
   - Все навигационные меню
   - Исключение: если это основная навигация по контенту страницы

2. **Роль = banner**
   - Вся шапка сайта
   - Логотипы, кнопки в хедере

3. **Роль = contentinfo**
   - Весь футер
   - Копирайт, нижние ссылки

4. **Роль = dialog и содержит "Закрыть"**
   - Модальные окна (если не нужны)
   - Попапы

5. **Размер изображения < 100x100 пикселей**
   - Иконки, декоративные элементы
   - Логотипы

6. **Текст содержит служебные слова:**
   - "Skip to main content"
   - "Закрыть диалоговое окно"
   - "Навигационное меню"
   - "Предыдущий слайд" / "Следующий слайд"
   - "Политика конфиденциальности" (если не нужна)

7. **Элемент находится вне main контента**
   - В header, footer, aside
   - В модальных окнах (если не нужны)

