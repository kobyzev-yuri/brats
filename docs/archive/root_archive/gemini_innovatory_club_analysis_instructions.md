# Инструкции для Gemini: Анализ страницы innovatory-club.ru/katalog/#service

## Задача

Проанализировать страницу каталога домов с отделкой и разбить её содержимое на смысловые блоки для дальнейшей обработки и импорта в базу знаний.

## URL страницы

```
https://innovatory-club.ru/katalog/#service
```

## Структура страницы

Страница содержит информацию о 4 типах отделки домов:
1. **BLACK BOX** (черновой ремонт)
2. **WHITE BOX** (предчистовой ремонт)
3. **STANDARD** (стандартный ремонт)
4. **DESIGN** (дизайнерский ремонт)

## Смысловые блоки для извлечения

### 1. Блок: Тип отделки (BLACK BOX / WHITE BOX / STANDARD / DESIGN)

**Что извлекать:**
- Заголовок типа отделки (например: "BLACK BOX (черновой ремонт)")
- Краткое описание под заголовком (если есть)

**Пример:**
```
BLACK BOX (черновой ремонт)
У вас есть возможность сделать ремонт вашей мечты
```

### 2. Блок: Описание типа отделки

**Что извлекать:**
- Основной текст описания типа отделки
- Характеристики и особенности
- Целевая аудитория (если упоминается)

**Пример для BLACK BOX:**
```
Дома с минимальной подготовкой для дальнейшего ремонта BLACK BOX — удобное решение для людей, желающих самостоятельно сделать ремонт в соответствии со своими пожеланиями и возможностями.
```

**Пример для STANDARD:**
```
Просторный, уютный дом с качественным ремонтом, в который можно заселиться сразу после покупки. Мы построили более 300 домов и точно знаем, как сделать ремонт так, чтобы он радовал вас долгие годы. STANDARD — проверенное временем решение, сочетающее практичность, долговечность и стиль: минималистичная классика, которая не выходит из моды.
```

### 3. Блок: Общая площадь

**Что извлекать:**
- Общая площадь дома
- Площадь по этажам (если указана)

**Пример:**
```
Общая площадь: 140 м² (1 этаж — 80 м², 2 этаж — 60 м²)
```

### 4. Блок: Что включает (список элементов отделки)

**Что извлекать:**
- Заголовок "Что включает?" или "что включает"
- Список элементов отделки с описаниями

**Пример для BLACK BOX:**
```
черновой ремонт включает:
- Возведённые стены и перегородки
- Подведённые коммуникации (без чистовой отделки)
- Голые бетонные стены и пол
- Установленные окна и входные двери
```

**Пример для WHITE BOX:**
```
предчистовой ремонт включает:
- Бетонная стяжка пола — ровное основание, готовое под укладку финишного покрытия: ламината, плитки или паркета.
- Выравнивание, грунтовка, штукатурка стен и потолков — поверхности подготовлены под покраску или поклейку обоев.
- Разводка электрики — установлены розетки, выключатели, выведены все необходимые провода.
- Разводка сантехники — проведены трубы водоснабжения и канализации, подготовлены точки подключения.
- Окна и двери — установлены энергосберегающие стеклопакеты и надёжная входная дверь.
- Отопление и коммуникации — смонтированы трубы, установлена система отопления и радиаторы.
```

**Пример для STANDARD:**
```
стандартный ремонт включает:
- Чистовая отделка — стены выровнены, оштукатурены, окрашены в выбранный цвет.
- Напольное покрытие — уложен ламинат или керамогранит на выбор.
- Отделка санузлов — стены и пол облицованы плиткой, установлена сантехника.
- Потолки — натяжные, аккуратные, эстетичные, выбранного дизайна.
- Электрика — установлены розетки, выключатели, освещение.
- Окна и двери — надёжные стеклопакеты, межкомнатные и входная дверь.
- Отопление и коммуникации — подключены трубы, радиаторы, котёл, водоснабжение, канализация.

На выбор доступны варианты отделки:
- 2 варианта керамогранита
- 6 видов ламината
- 6 оттенков окраски стен
- 50 вариантов натяжных потолков
- 6 видов плитки для санузла
```

**Пример для DESIGN:**
```
дизайнерский ремонт включает:
- Дизайн-проект — выбор из 3 готовых вариантов, продуманных до мелочей: от расстановки мебели до освещения.
- Дизайнерский ремонт — выполнен мастерами с многолетним опытом, в точном соответсвии с дизайн проектом. Для ремонта используются только проверенные высококачественные материалы и технологии, проверенные временем.
- Мебель и декор — дизайнерские люстры, шкафы, полки и стеновые панели согласно дизайн-проекту
```

### 5. Блок: Стоимость

**Что извлекать:**
- Базовая стоимость дома
- Стоимость ремонта (если указана отдельно)
- Примечания о расчете стоимости

**Примеры:**
```
BLACK BOX: 8 350 000 Р
WHITE BOX: 8 950 000 Р
STANDARD: 8'950'000 Р + ремонт-700'000 Р*
DESIGN: 8'950'000 Р + дизайнерский ремонт-1'500'000 Р*

*ОБРАЩАЕМ ВНИМАНИЕ! Точную стоимость ремонта рассчитаем индивидуально при обращении в отдел продаж по актуальным тарифам во время обращения.
```

### 6. Блок: Преимущества поселка

**Что извлекать:**
- Заголовок "Преимущества"
- Список преимуществ с описаниями

**Пример:**
```
Преимущества:
- Высокое качество — строительных материалов, надежная электрика, современные экологичная технология утепления дома.
- важные системы — Все дома оснащены инновационной автономной энергоэффективной системой отопления воздух/вода.
- внешняя красота — На всех домах поселка установлена фасадная декоративная подсветка фронтонов и французский балкон
- территория дома — Благоустроенная придомовая территория: красивый забор, откатные ворота, калитка на замке
- зоны отдыха — Во всех дворах есть достаточно место под баню, беседку, сад, мангал, выводы электричества, освещение.
- функциональный двор — Полностью благоустроенный красивый двор: парковочные места, кованное крыльцо, отводы воды, чернозем
```

### 7. Блок: Планировки

**Что извлекать:**
- Заголовок "Планировки" или "Планировки"
- Описание вариантов планировок
- Информация о вариантах дизайнерского ремонта (если есть)

**Пример:**
```
Планировки:
большой выбор

ВАРИАНТЫ дизайнерского ремонта:
- Вариант 1: первый этаж
- Вариант 2: первый этаж и лестница
- Второй этаж: комнаты и санузлы
```

### 8. Блок: Контакты

**Что извлекать:**
- Адрес
- Телефон
- Email
- График работы/просмотра

**Пример:**
```
АДРЕС: г. Краснодар, хутор Октябрьский, улица Гренадерская, дом 10/2

График просмотра домов: ежедневно с 10:00 до 17:00

Телефон: +7 (988) 199-89-98
Email: innovatory-club@yandex.ru
```

### 9. Блок: Галерея изображений

**Что извлекать:**
- Только контентные изображения (размер > 100x100px)
- Описание изображений (alt-текст, если есть)
- URL изображений

**Игнорировать:**
- Логотипы
- Иконки (< 100x100px)
- Декоративные элементы

### 10. Блок: Дополнительная информация

**Что извлекать:**
- Информация о застройщике (если есть на странице)
- Информация о поселке (если есть)
- Любая другая релевантная информация для покупателя

- Верхнее меню с ссылками (О ПОСЕЛКЕ, КОТТЕДЖИ, О ЗАСТРОЙЩИКЕ, КОНТАКТЫ)
- Боковое меню (если есть)
- Хлебные крошки

### 2. Хедер
- Логотип "ИННОВАТОРЫ КЛУБ"
- Кнопки "ОТПРАВИТЬ ЗАПРОС", "В КАТАЛОГ"
- Телефон в шапке (если дублируется в контактах)

### 3. Футер
- Нижнее меню
- Копирайт "©2025 КП «Инноваторы клуб»"
- Ссылки на политику конфиденциальности

- Поля ввода (имя, телефон, чекбокс согласия)

**Примечание:** Формы можно извлекать, если нужна информация о структуре форм для интеграции, но обычно их игнорируют при извлечении контента.

### 5. Декоративные элементы
- Иконки (< 100x100px)
- Декоративные изображения
- Фоновые элементы
- Кнопки навигации ("Предыдущий слайд", "Следующий слайд", "Закрыть")

### 6. Служебные тексты
- "Skip to main content"
- "Закрыть диалоговое окно"
- "Навигационное меню"
- Тексты в формах (если не нужны)

## Правила фильтрации

### По ролям элементов (ARIA roles):
- ✅ **Извлекать из:** `role: main`, `role: article`, `role: section`
- ❌ **Игнорировать:** `role: navigation`, `role: banner`, `role: contentinfo`, `role: dialog` (если не нужны формы)

### По размеру изображений:
- ✅ **Извлекать:** изображения > 100x100px (контентные фото)
- ❌ **Игнорировать:** изображения < 100x100px (иконки, логотипы)

### По расположению:
- ✅ **Извлекать:** контент из основного блока страницы
- ❌ **Игнорировать:** элементы из header, footer, aside, навигации

### По содержимому:
- ✅ **Извлекать:** тексты с информацией о домах, отделке, ценах, контактах
- ❌ **Игнорировать:** служебные тексты, тексты кнопок навигации, тексты форм (если не нужны)
