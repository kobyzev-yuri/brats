## 📋 Быстрый доступ к ключевым файлам

- **Запуск**: `./start.sh` или `uvicorn api.main:app --host 0.0.0.0 --port 8001`
- **Конфигурация**: `config.env` (скопировать из `config.env.example`)
- **API документация**: http://localhost:8001/docs
- **Основной сервис**: `services/kb_service.py`
- **API endpoints**: `api/main.py`

## 🔗 Связь с проектом

- **База данных**: PostgreSQL с pgvector (база brats, config.env в корне проекта)
- **Таблица**: `knowledge_base` (схема из `docs/KB_STRUCTURE.md`)
- **Интеграция**: Через n8n HTTP Request nodes (см. `N8N_INTEGRATION.md`)
- **Агент продаж**: Подключается через n8n workflow

## ⚠️ Важные замечания

1. **PostgreSQL + pgvector** должны быть установлены и настроены
2. **OpenAI API ключ** необходим для генерации embeddings
3. **Таблица knowledge_base** должна быть создана согласно схеме из README.md
4. **Конфигурация** настраивается через `config.env`

## 📝 Команды для восстановления контекста

```bash
# Активация окружения
cd /projects/brats
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Настройка конфигурации
cd kb-service
cp config.env.example config.env
# Отредактировать config.env

# Запуск
./start.sh
```
