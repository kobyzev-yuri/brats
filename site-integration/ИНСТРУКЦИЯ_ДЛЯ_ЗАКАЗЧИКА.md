# Инструкция для заказчика: Интеграция нейропродажника на сайт

## 🎯 Что нужно сделать

Вставить JavaScript код на сайт для автоматической инициации нейропродажника при вводе телефона и email в формах.

---

## 📋 Шаг 1: Получить код

Код находится в файле: `site-integration/SITE_INTEGRATION_CODE.md`

Или используйте упрощенную версию ниже.

---

## 📋 Шаг 2: Настроить URL webhook

Перед вставкой кода на сайт, **обязательно замените** URL webhook на реальный:

```javascript
// НАЙДИТЕ ЭТУ СТРОКУ В КОДЕ:
n8nWebhookUrl: 'https://your-n8n-domain.com/webhook/sales-agent-kb',

// ЗАМЕНИТЕ НА РЕАЛЬНЫЙ URL:
n8nWebhookUrl: 'https://n8n.yourdomain.com/webhook/sales-agent-kb',
```

**Где взять URL webhook:**
1. Откройте n8n
2. Найдите workflow "Sales Agent - KB Integration"
3. Откройте узел "Webhook"
4. Скопируйте Production URL (например: `http://localhost:5678/webhook/sales-agent-kb`)

## 📋 Шаг 4: Добавить атрибуты к формам (опционально, но рекомендуется)

Для лучшей идентификации форм добавьте атрибут `data-form-type` к каждой форме:

```html
<!-- Форма резервирования BLACK BOX -->
<form data-form-type="reservation_black_box">
    <input type="tel" name="phone" placeholder="Телефон">
    <input type="email" name="email" placeholder="Email">
    <button type="submit">Отправить</button>
</form>

<!-- Форма резервирования WHITE BOX -->
<form data-form-type="reservation_white_box">
    <input type="tel" name="phone" placeholder="Телефон">
    <input type="email" name="email" placeholder="Email">
    <button type="submit">Отправить</button>
</form>

<!-- Калькулятор ипотеки -->
<form data-form-type="mortgage_calculator">
    <input type="tel" name="phone" placeholder="Телефон">
    <input type="email" name="email" placeholder="Email">
    <button type="submit">Рассчитать</button>
</form>
```

**Возможные значения `data-form-type`:**
- `reservation_black_box` - Резервирование BLACK BOX
- `reservation_white_box` - Резервирование WHITE BOX
- `reservation_standard` - Резервирование STANDARD
- `reservation_design` - Резервирование DESIGN
- `mortgage_calculator` - Калькулятор ипотеки
- `viewing_appointment` - Запись на просмотр
- `contact_form` - Контактная форма

## ✅ Что происходит после установки

1. **При вводе телефона** (минимум 10 цифр):
   - Автоматически отправляется событие в n8n
   - Нейропродажник получает уведомление
   - Событие также отправляется в Яндекс.Метрику (если настроена)

2. **При вводе email** (валидный формат):
   - Автоматически отправляется событие в n8n
   - Нейропродажник получает уведомление
   - Событие также отправляется в Яндекс.Метрику (если настроена)

3. **При отправке формы**:
   - Отправляется финальное событие с полными данными
   - Нейропродажник может инициировать диалог

4. **При клике по телефону** (`<a href="tel:+79981234567">`):
   - Отправляется событие "phone_click"
   - Это критическое событие для инициации агента

5. **При клике по WhatsApp**:
   - Отправляется событие "whatsapp_click"
   - Это критическое событие для инициации агента

## 🧪 Проверка работы

### 1. Откройте консоль браузера

1. Нажмите `F12` или `Ctrl+Shift+I` (Windows/Linux) / `Cmd+Option+I` (Mac)
2. Перейдите на вкладку **Console**

### 2. Введите телефон или email в форму

1. Откройте любую форму на сайте
2. Начните вводить телефон (например: `+7 999 123-45-67`)
3. Подождите 1 секунду после ввода

### 3. Проверьте Network запросы

1. В консоли перейдите на вкладку **Network**
2. Найдите запрос к вашему n8n webhook
3. Проверьте, что запрос успешный (статус 200)

### 4. Проверьте в n8n

1. Откройте n8n
2. Перейдите в **Executions**
3. Найдите последнее выполнение workflow
4. Проверьте, что данные пришли корректно

## ❓ Часто задаваемые вопросы

### Q: Нужно ли менять существующие формы?

**A:** Нет, код автоматически найдет все поля телефона и email на странице. Но рекомендуется добавить атрибут `data-form-type` для лучшей идентификации.

### Q: Будет ли работать на мобильных устройствах?

**A:** Да, код полностью совместим с мобильными устройствами.

### Q: Что если форма загружается динамически (через AJAX)?

**A:** Код автоматически отслеживает новые элементы на странице и будет работать с динамическими формами.

### Q: Безопасно ли отправлять телефон и email?

**A:** Код отправляет данные через HTTPS. Для дополнительной безопасности можно использовать хеширование (см. раздел "Безопасность" в `SITE_INTEGRATION_CODE.md`).

### Q: Можно ли отключить отправку в Яндекс.Метрику?

**A:** Да, установите `enableYandexMetrika: false` в конфигурации.

### Q: Что делать, если код не работает?

**A:** 
1. Проверьте консоль браузера на ошибки
2. Убедитесь, что URL webhook правильный
3. Проверьте, что workflow опубликован в n8n
4. Проверьте CORS настройки (если webhook на другом домене)
